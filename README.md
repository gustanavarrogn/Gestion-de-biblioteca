# Sistema de Gestión de Biblioteca

## Descripción

Aplicación de escritorio desarrollada en Java utilizando NetBeans y Swing para la gestión básica de libros en una biblioteca. Permite agregar libros y visualizarlos en una tabla mediante una interfaz gráfica.

## Instalación

1. Clonar el repositorio desde GitHub:
   git clone https://github.com/gustanavarrogn/Gestion-de-biblioteca.git

2. Abrir el proyecto en Apache NetBeans.

3. Ejecutar el archivo:
   MenuPrincipal.java

## Uso

1. Ingresar el **Título** del libro.
2. Ingresar el **Autor**.
3. Ingresar el **Año** de publicación.
4. Presionar **Agregar Libro**.
5. El libro aparecerá en la tabla de registros.

## Guía de Despliegue

1. Abrir el proyecto en NetBeans.
2. Compilar el proyecto.
3. Ejecutar la aplicación desde la clase principal `MenuPrincipal`.
4. La interfaz gráfica permitirá gestionar los libros de forma sencilla.

## Tecnologías utilizadas

* Java
* Swing
* NetBeans IDE
* Git / GitHub
